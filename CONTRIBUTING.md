# Contributing to Trale

Thanks for your interest in contributing! This is a small project, and contributions of any kind are welcome.

## Quick Start

1. Fork and clone the repository
2. Follow setup in [README.md](README.md) - don't forget `lake exe cache get`!
3. Build with `lake build`

## How to Contribute

**Found a bug?** Open an issue describing what you found and how to reproduce it.

**Have an idea?** Open an issue to discuss it - I'd love to hear your thoughts!

**Want to contribute code?**
1. Create a branch for your changes
2. Make your changes - try to follow the existing code style
3. Test with `lake build TraleTest`
4. Open a pull request

**Documentation improvements** are always welcome - typos, clarifications, examples, anything helps!

## Project Structure

- `Trale/` - Main library code
- `TraleTest/Transfer/` and `TraleTest/Lemmas/` - Examples and tests
- `TraleTest/Research/` - Experimental code
- `docs/` - Documentation

## Questions?

Just open an issue! I'm happy to help.
